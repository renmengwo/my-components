# Button 按钮

`Button` 组件用于触发操作。

## Vue 版本

### 效果展示

<VueButton />

### 使用方法

```vue
<template>
  <Button>点击我</Button>
</template>

<script setup>
import { Button } from 'vue-components';
</script>
```
## react 版本

<ReactButton />

```jsx  
import { Button } from 'react-components';
function App() {
    return <Button></Button>;
}


```
